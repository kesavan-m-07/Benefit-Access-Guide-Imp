# Benefit Access Guide

A React application built with modern tooling and best practices for scalable, type-safe, and maintainable development.

## Tech Stack

* React
* TypeScript
* TanStack Router
* React Hook Form
* Zod
* Tailwind CSS
* Framer Motion
* pnpm

## Prerequisites

* Node.js 20+
* pnpm 9+

Verify your installation:

```bash
node -v
pnpm -v
```

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd benefit-access-guide
```

Install dependencies:

```bash
pnpm install
```

## Running the Application

Start the development server:

```bash
pnpm dev
```

## Production Build

Create a production build:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
